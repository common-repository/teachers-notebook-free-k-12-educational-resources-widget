Plugin to add a Teachers Notebook FREE K-12 Educational Resources widget to the sidebar or footer, or embed into a page or post using shortcodes.

== Description ==

Plugin to add a [Teachers Notebook](http://teachersnotebook.com) **FREE** K-12 Educational Resources widget to the sidebar or footer, or embed into a page or post using shortcodes.  Once installed and activated, the widget will display a few of the thousands of **FREE** K-12 Educational Resources for the selected grade level and category which are available for download from the Teachers Notebook website.

Note: A Teachers Notebook member account IS NOT REQUIRED.  Information to be displayed for the selected grade level and category will be retrieved anonymously from the Teachers Notebook website.  For more information on Teachers Notebook membership, please visit: http://www.TeachersNotebook.com.

== Installation ==

Install the plugin:

1. Upload the plugin directory to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress

To add a widget to the sidebar or footer:

1. From the Dashboard -> Widgets page, drag and drop the TN Free Widget to the desired location sidebar or footer location.
1. Select the Grade Level and the Category of educational resources to display as the default.
1. Click the Save button to save the configuration.

For shortcode usage:

1. Insert the shortcode [teachers_notebook_free_widget] into the page or post content.

Note: A Teachers Notebook member account IS NOT REQUIRED.  Information to be displayed for the selected grade level and category will be retrieved anonymously from the Teachers Notebook website.  For more information on Teachers Notebook membership, please visit: http://www.TeachersNotebook.com.

== Frequently Asked Questions ==

= Is Teachers Notebook membership required? =

No.  As the description *Teachers Notebook **FREE** K-12 Educational Resources* implies, the widget will display a few of the thousands of **FREE** K-12 Educational Resources available for download from the Teachers Notebook website.  A link is also provided for direct access to the Teachers Notebook website page where all of the **FREE** K-12 Educational Resources available for download can be searched and reviewed.

= Where does the widget get its data? =

The appropriate data for the selected grade level and category is retrieved anonymously from the Teachers Notebook website.
